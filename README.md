# 🧪 Playwright Test Automation Suite

End-to-end test automation project built with **Playwright** and **TypeScript**, demonstrating clean test architecture and real-world QA practices.

---

## 🚀 Overview

This project showcases a structured approach to testing modern web applications using:

- Page Object Model (POM)
- End-to-End (E2E) testing
- UI validation and user flows
- Reusable test setup and test data

The tests are implemented against a demo application to simulate real-world QA scenarios.

---

## 🛠 Tech Stack

- Playwright
- TypeScript
- Node.js

---

## 🔍 Test Coverage

### 🔐 Authentication

- Successful login with valid credentials
- Error handling for invalid login
- Validation for empty input fields

### 🧭 UI & Navigation

- Page load verification
- Inventory page validation
- Element visibility checks

### 🛒 E2E User Flow

- Add product to cart
- Cart badge validation
- Cart page verification
